Border Width

Customizable in variable `$border-width-classes`. Every item in the list will add many classes to cover all directions and responsive. `b-x`, `bt-x`, `bb-x`, `bl-x`, and `br-x`.

Border Style

Must be used in conjuction with Border Width class.

Border Color

Customizable in variable $border-color-classes. Must be used in conjuction with Border Width class.
