Font Sizes

Customizable in variable `$font-size-classes`. It accepts map and pixel. Providing a map will create responsive font sizes. It must contain "base" key, followed by the screen sizes breakpoint. For example: `(base: 26px, medium: 22px, small: 18px, mini: 16px)`
