Grid Mixins

``` grid() ``` Grid mixin should only be applied on `` element. The parameters in order are: - `$column` (int) - Total number of columns per row. - `$gap` (px) - Distance between columns. - `$rim` (px) - Distance from edge of screen. - `$width` (px) - Grid's max width. - `$collapse` (bool) - Clear the gap or not. Default: false. ``` column() ``` Column parameters in order are: - `$span` (int) - Column size for large screen >768px - `$span-s` (int) - Column size for small screen <=768px - `$span-xs` (int) - Column size for extra small screen <=480px **EXAMPLE** ```html
``` ```scss .my-grid +grid( $columns:10, $gap:40px ) .sidebar +column(3, 4, 10) // 3 on Large, 4 on Small, 10 on XSmall screen .main +column(7, 6, 10) ```
