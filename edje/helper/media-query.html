Media Query

We provide variables that can be used in conjunction with `@media`. ``` @media ($below-s) { // styling below small screen } @media ($below-xs) and ($landscape) { // ... } ```

Customization

To change the size of a keyword, such as `s`, edit the variable `$size-s` in settings. If you want to add your own media-query size, simply create new variable in `_settings.scss`. ``` $below-watch : "max-width: 220px"; $above-watch : "min-width: 221px"; ``` ``` @media ($below-watch) { ... } ``` -----

One-Time Use

When you only need the media-query size once, we provide a function so you don't need to create new variable for it.

Print and Screen

There is no need for fancy variables when specifying Print / Screen media query. ``` @media only print { ... } @media only screen and ($below-s) { ... } ```
