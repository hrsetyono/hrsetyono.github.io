Edje Color

Any colors defined in `$h-colors` (in `_settings.scss`) will be available to use as CSS Variable. For example: ```sass $h-colors: ( 'red': #d32f2f, 'green': #4caf50, 'blue': #1976d2, 'main': var(--blue), .... ); ``` It will output each item into `:root` and it also add the RGB version: ```css :root { --red: #d32f2f; --redRGB: 211, 47, 47 --green: #4caf50; --greenRGB: 76, 175, 80 --blue: #1976d2; --blueRGB: 25, 118, 210; --main: var(--blue); --mainRGB: var(--blueRGB); ... } ```

Color Properties

Properties that accept color values are: - `color` shortened to `c` - `background-color` shortened to `bg` - `border-color` shortened to `bc` Combine those properties with color values defined above. You can add special syntax for transparency too: ```sass +h( c-red bg-black_50 ) // Output color: var(--red); background-color: rgba(var(--blackRGB), .50); ``` If the key is not found, it will output it as is: ```sass +h( c-inherit ) // Output color: inherit ```
