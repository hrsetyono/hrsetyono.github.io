Customization

Across the documentation we mentioned whether it is customizable via variable or not. When it does, simply open `_settings.scss`, find the variable, and edit it. **EXAMPLE:** We want to add new font-family "Papyrus". The shorthand is going to be `ff-script`. As written in the documentation, we need to find `$h-font-family`. Then we add new item in the "values" map: ```scss $h-font-family: ( names: ( ff-: font-family ), values: ( body: $font-family, header: $font-family-header, script: "Papyrus" ) ); ``` Now you can use `ff-script` to apply Papyrus font: ```scss .page-title +h( ff-script ) ``` -----

Direct Format

Some of the variables don't use Name-Value pair, instead it directly specify the Class name and its CSS style like in `$h-border-radius`: ```scss $h-border-radius: ( br: ( border-radius: $g-radius ), br-circle: ( border-radius: 50% ), br-pill: ( border-radius: 999px ), ); ``` -----

Pseudoselector Format

You can see the example of pseudoselector format in `$h-hover` variable: ```scss $h-hover: ( hover-dim: ( ':hover': ( opacity: .6 ), ':active': ( opacity: .8 ), ), hover-grow: ( ':hover': ( '-webkit-transform': scale(1.05), transform: scale(1.05) ), ':active': ( '-webkit-transform': scale(1), transform: scale(1) ), ), hover-u: ( ':hover': ( text-decoration: underline ) ), ); ```

Extend Format

You can see the example of extending format in `$h-font-size` variable: ```scss $h-font-size: ( names: ( 'fs-': font-size ), values: ( h1: '%h1', h2: '%h2', ... ) ); ``` When a value is prefixed with percent sign (%), it will trigger `@extend` command. For Heading 1 to 6, we already wrote the base styles in the framework.
