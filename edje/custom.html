Customization

Across the documentation we mentioned whether it is customizable via variable or not. When it does, simply open `_settings.scss`, uncomment it, and add extra item to the list. **Note**: Since the list might refer to a variable, you need to uncomment that variable too. **EXAMPLE:** We want to add a new font family. As written in the documentation, we need to find and override `$h-font-family`. Find this variable in settings: ``` scss // $h-font-family: ( // names: ( ff-: font-family ), // values: ( // body: $font-family, // header: $font-family-header // ) //); ``` **Uncomment it**, and add extra item to `values` argument: ```scss $h-font-family: ( names: ( ff-: font-family ), values: ( body: $font-family, header: $font-family-header, script: "Papyrus" ) ); ``` Now you can use `ff-script` to apply Papyrus font: ```scss .page-title { @include h( ff-script ); } ``` Don't like the shorthand `ff-` for font-family? You can simply change it in `names` argument! -----

Direct Format

Some of the variables don't use Name-Value pair, instead it directly specify the Class name and its CSS style like in `$h-border-radius`: ```scss $h-border-radius: ( br: ( border-radius: $g-radius ), br-circle: ( border-radius: 50% ), br-pill: ( border-radius: 999px ), ); ``` -----

Pseudoselector Format

You can see the example of pseudoselector format in `$h-hover` variable: ```scss $h-hover: ( hover-dim: ( ':hover': ( opacity: .6 ), ':active': ( opacity: .8 ), ), hover-grow: ( ':hover': ( '-webkit-transform': scale(1.05), transform: scale(1.05) ), ':active': ( '-webkit-transform': scale(1), transform: scale(1) ), ), hover-u: ( ':hover': ( text-decoration: underline ) ), ); ```

Extend Format

You can see the example of extending format in `$h-font-size` variable: ```scss $h-font-size: ( names: ( 'fs-': font-size ), values: ( h1: '%h1', h2: '%h2', ... ) ); ``` When a value is prefixed with percent sign (%), it will trigger `@extend` command. In the case of Heading 1 to 6, we already define the style within the framework. If you want to create a new one, add its definition at the **bottom of settings**: **NOTE**: In settings, you can't use any helper functions ```scss ... %subtitle { font-size: 1.25em; @media ($below-s) { font-size: 1.125em; } @media ($below-xs) { font-size: 1em; } } ```
