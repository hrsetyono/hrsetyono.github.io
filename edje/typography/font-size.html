Font Sizes

Customizable in variable `$h-font-size`.
The Header's font size extends a style declared in the framework. Its sizing can be customized in variable `$h1` to `$h6`. Those variables follows a special format to allow responsive sizing. ```scss $h1: (base: 42px, s: 30px); ``` The value above means H1 is 42px at normal and become 30px when below small (s) screen. You can also use the keyword `m`, `xs`, and `l` here: ```scss $h1: (base: 42px, m: 36px, s: 30px, xs: 26px); ```
