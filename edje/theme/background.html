Background Color

Customizable in variable `$h-background-color`.

Background Image

Customizable in variable `$h-background-image`. By default this variable is **empty**. You can put either `url()` or any gradient value like `linear-gradient()`. **EXAMPLE:** ```scss $h-background-image: ( names: ( bg-: background-image ), values: ( pattern1: url('../images/pattern1.png'), gradient1: linear-gradient( to top, #333, #333 50%, #eee 75%, #333 75%) ) ) !default; ``` Following the snippet above, the class name to use will be `bg-pattern1` and `bg-gradient1`.
