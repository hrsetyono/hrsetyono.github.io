Border Width

Customizable in variable `$h-border-width`. Every item in the list will add many classes to cover all directions `bx`, `btx`, `bbx`, `blx`, and `brx`.

Border Style

Must be used in conjuction with Border Width class.

Border Color

Customizable in variable `$h-border-color`. Must be used in conjuction with Border Width class.

Border Radius

Customizable in variable `$h-border-radius`.
